# NodeBack
Simple todolist Application - for future wrapping with web view - Android.

Create,Read, Update and Delete notes.

**No extra design at all, just the functionality to demonstrate capability.

Front:
1. HTML + CSS.
2. Javascript:
    2.1 Vanilla.
    2.2 JQuery.

Back:
1. NodeJS:
    1.1 Express.
2. Mongo (Mongoose).
